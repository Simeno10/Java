# Java Repository

Collection of Java projects and exercises covering:

## Features

- Employee Management System (CRUD)
- Spring Boot REST API
- Calculator
- Password Checker
- Java Collections
- Unit Testing (JUnit)
- Text Analyzer

## Technologies

- Java
- Spring Boot
- Maven
- JPA
- JUnit

## Example Projects

### Calculator
Performs basic arithmetic operations.

### Password Checker
Validates password strength based on selected rules.

### Employee Management System
REST API for managing employees using Spring Boot.

### Text Analyzer
Analyzes text and provides:
- Word frequency count
- Most common word
- Character count

## How to Run

```bash
mvn clean install
mvn spring-boot:run
